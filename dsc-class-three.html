Class 3 Digital Signature Certificate — Enhanced Security

Class 3 DSC provides the highest level of identity assurance. It is mandatory for government e-procurement, e-tendering, and high-security transactions where Class 2 is not accepted.

Where Class 3 is mandatory: GeM (Government e-Marketplace) for procurement above ₹25,000. IREPS (Indian Railways Electronic Procurement System). Central Public Procurement Portal (CPP). Various state e-procurement portals. Defence procurement portals.

In-person verification: Unlike Class 2, Class 3 requires the applicant to appear before the Certifying Authority (or authorised agent) with original documents. When Is Class 3 Digital Signature Certificate Not Required?

A Class 3 DSC may not be required when: (a) you are not a company director, authorised signatory, or professional required to file statutory returns, (b) an existing valid DSC covers all your filing requirements, (c) the entity is a sole proprietorship with no digital filing obligations, or (d) you have delegated all filing responsibilities to a professional. Class 3 DSC is mandatory for MCA filings, GST return signing (for companies), and income tax return signing for entities requiring audit.

Frequently Asked Questions

When is Class 3 needed instead of Class 2?

For government e-tendering portals (GeM, CPPP, IREPS) and e-procurement. If you bid on government contracts, you need Class 3.

What is Class 3 DSC used for?

Government e-tendering (GeM, IREPS, CPP Portal), high-security e-procurement, and applications requiring enhanced identity assurance.

How is Class 3 different from Class 2?

Class 3 requires mandatory in-person verification (not video KYC). Highest assurance level. Cost: Rs 2,000-3,500. Processing: 2-5 days.

Which portals require Class 3?

GeM (Government e-Marketplace), IREPS (Indian Railways), CPP Portal, various state e-procurement portals, and defence procurement.

Can I upgrade Class 2 to Class 3?

No direct upgrade. Must purchase separate Class 3 DSC. Both can coexist for same person on different tokens.

Is Class 3 needed for company registration?

No. Class 2 sufficient for MCA, IT, GST. Class 3 only for e-tendering and specific high-security government portals.
